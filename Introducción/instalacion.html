# Instalación

# Instalando Node.js y NPM

Antes de instalar Hapi.js es necesario contar con Node asi como Npm que es el gestor de paquetes por defecto para Node.js, un entorno de ejecución para JavaScript.

Instalación de Node.js

sudo apt-get update
sudo apt-get install node

Instalación de NPM

sudo apt-get install npm

Nota

Para verificar una correcta instalación de los paquetes ejecutamos:

node -v
npm -v

# Instalando Hapi.js

Iniciar un proyecto de Node.js

Para utilizar Hapi.js en un proyecto de Node creamos uno nuevo con el comando:

sudo npm init

Instalación de Hapi

sudo npm i hapi -S

Nota

-S al ser una dependencia del proyecto y no de desarrollo
